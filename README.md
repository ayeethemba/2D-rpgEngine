# That Time I Was an Office Worker and Put into a Coma by a Demon Sleep God

A 2D side-scrolling RPG built with JavaScript and p5.js for CS345 (Software Engineering) at James Madison University, Spring 2026.

## About

You play as an office worker who gets pulled into a coma by a demon sleep god. Fight through levels, interact with NPCs through a dialogue system, and battle enemies to find your way out.

## Tech Stack

- JavaScript (p5.js)
- HTML5 / CSS3
- Git / GitHub (133 commits across a team of 5)

## Features

- 2D side-scrolling gameplay with player movement and collision
- Enemy AI with custom behavior logic
- Dialogue system for NPC interactions
- Multiple levels, including a boss level
- Custom sprites and sound effects

## How to Run

1. Clone the repository: 
   git clone https://github.com/chaoaj/sp26_cs345_team03.git
2. Open index.html in your browser
3. No dependencies or installs required
